# AngularDD

A custom designed &lt;select> dropdown using Angular and Sass 

## Installation

Clone repo locally using

```
$ git clone [repo ssh clone url]
```

Using Bower run

```
$ bower install
```

Make sure you compile the sass file using

```
sass src/style.scss
```

# Development

```
sass --watch src/style.scss
```
